# ☕ Cafe Cozy

> A warm, inviting online presence for your favorite neighborhood café.

## 📖 About

**Cafe Cozy** is a modern, responsive website for a cozy café experience — bringing the warmth of your local coffee shop to the web. Whether you're browsing the menu, exploring our story, or making a reservation, Cafe Cozy makes it effortless and delightful.

---

## ✨ Features

- 🏠 **Home Page** — Welcoming hero section with featured items and ambiance
- 📋 **Menu** — Full food & beverage menu with categories and descriptions
- 📸 **Gallery** — Visual showcase of the café's atmosphere and dishes
- 📍 **About Us** — Our story, team, and values
- 📅 **Reservations** — Easy table booking form
- 📬 **Contact** — Location, hours, and contact form
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- [Git](https://git-scm.com/) installed on your machine
- *(Optional)* [VS Code](https://code.visualstudio.com/) with Live Server extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cafe-cozy.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd cafe-cozy
   ```

3. **Open in browser**
   ```bash
   # Option 1: Open directly
   open index.html

   # Option 2: Use VS Code Live Server
   # Right-click index.html → "Open with Live Server"
   ```

## 🚢 Deploying to GitHub Pages

1. Push your code to GitHub
2. Go to your repository → **Settings**
3. Navigate to **Pages** (in the left sidebar)
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save**
6. Your site will be live at:
   ```
   https://your-username.github.io/cafe-cozy
   ```
